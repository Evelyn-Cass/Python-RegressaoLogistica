# 🚢 Python-RegressaoLogistica

A Python project for performing logistic regression analysis, built with Jupyter Notebook. Demonstrates how to solve binary classification problems using the Titanic dataset.

## 📱 About The Project

Python-RegressaoLogistica is a simple and practical implementation of logistic regression. The project predicts passenger survival on the Titanic based on features like age, gender, and passenger class.

This project was created for study purposes and to demonstrate logistic regression (classification) using Python.

## ✨ Features

- Logistic regression model implementation
- Binary classification (survival prediction)
- Titanic dataset example
